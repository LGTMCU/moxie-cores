Memory Map
============

The Marin SoC has on-chip ROM and RAM, off chip RAM support, and a
number of memory mapped devices.

|------------+---------+-------------------+-------------------------------|
| 0x00001000 | 1k      | on-chip ROM       | bootloader                    |
|------------+---------+-------------------+-------------------------------|
| 0x10000000 | 16k     | on-chip RAM       | data/heap/stack for bootloads |
|------------+---------+-------------------+-------------------------------|
| 0x30000000 | 16MB    | off-chip RAM      | user code                     |
|------------+---------+-------------------+-------------------------------|
| 0xF0000000 | 8 bytes | 7-segment display |                               |
|------------+---------+-------------------+-------------------------------|
| 0xF0000008 | 8 bytes | UART              |                               |
|------------+---------+-------------------+-------------------------------|
| 0xF0000010 | 8 bytes | PIC               |                               |
|------------+---------+-------------------+-------------------------------|
| 0xF0000020 | 8 bytes | Bus Watchdog      |                               |
|------------+---------+-------------------+-------------------------------|
